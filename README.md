## Run Locally

Clone the project

```bash
  git clone https://github.com/callbomber
```

Go to the project directory

```bash
  cd callbomber
```

Install dependencies

```bash
  apt install python3

```
```bash
  pip install requests pyrogram tgcrypto user_agent
```


Start the server

```bash
  python3 sms.pyc
```
